- CSS
- HTML
- JavaScript

## Features
- Active Notes
- Archive Notes
- Search Notes
- Add Note
- Delete Note

## Requirement
- Node.js >= 14

## Installation
- Clone Repo
  ```sh
  https://github.com/cimotty/personal-notes.git
  ```
- Install Dependencies
  ```sh
  npm install
  ```
- Running App
  ```sh
  npm run dev
  ```
